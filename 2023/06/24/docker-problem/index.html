本博客记录在学习docker过程中遇到的一些问题

目录挂载

docker run -it -v /Users:/usr/local/tomcat/webapps –name=mew-tomcat my-tomcat

这将把本地的Users文件夹下面的目录挂载到/usr/local/tomcat/webapps文件夹中，并且将里面原来的东西进行覆盖

之后两遍数据双向绑定，修改一边，另外一边同步修改

  • scratch image

scratch 可能是docker中最小的镜像。事实上，这个镜像是空的，里面不含有任何文件或者文件夹，它仅仅是构建我们镜像的一个起始点。

为了在scratch上面运行二进制文件，我们的可执行程序需要静态编译和自包含。意味着图像中没有编译器，只剩下系统调用。

你也不能够登陆进入容器，毫无疑问，其中也没有shell，除非你明确增加一个。
