本博客将会记录一些Hexo配置过程中遇到的问题

图片上传插件

可以使用插件推荐安装插件：hexo-image-link
安装方法：

npm install hexo-image-link –save

修改config.yml配置文件：

hexo-asset-folder: true

这样设置之后每次hexo new “title”就能生成title.md和title文件夹
接着将图片拷贝到同名文件夹下面
markdown语法为：

! [ 配置文件 ] (/image.jpg)
